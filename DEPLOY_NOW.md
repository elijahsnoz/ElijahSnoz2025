## 🚀 Deploy in 3 Steps (Netlify - Recommended)

### Step 1: Push to GitHub

```bash
cd /Users/xworld/Desktop/PROGRAMMING/PROGRAMING/alx/ElijahSnoz2025

# Initialize git (if not already done)
git init

# Add all files
git add .

# Commit
git commit -m "feat: mobile-optimized portfolio with analytics and Instagram feed"

# Create GitHub repo and push
git remote add origin https://github.com/elijahsnoz/portfolio.git
git branch -M main
git push -u origin main
```

### Step 2: Deploy to Netlify

1. Go to **https://netlify.com**
2. Click **"Add new site"** → **"Import an existing project"**
3. Choose **"Deploy with GitHub"**
4. Select your repository: `elijahsnoz/portfolio`
5. Click **"Deploy site"**

**Done!** You'll get a URL like: `elijahsnoz.netlify.app`

### Step 3: Custom Domain (Optional)

1. Buy domain (Namecheap, GoDaddy, etc.)
2. In Netlify: **Domain settings** → **Add custom domain**
3. Update DNS records
4. Wait 24 hours for DNS propagation

## 🐛 Troubleshooting

### "Images not loading on deployed site"
- Check file paths are correct
- Ensure images folder uploaded
- Try lowercase filenames

### "Analytics not tracking"
- Clear browser cache
- Check localStorage enabled
- Visit site in incognito

### "Instagram not showing"
- Wait 3-5 seconds for embed
- Check internet connection
- Try different browser
